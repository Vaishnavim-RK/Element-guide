HTML is the standard language used to structure web pages and web applications. In this project, it provides the backbone of the interactive periodic table by defining the layout, elements, and containers.
o	Creates the main structure of the periodic table with rows, columns, and individual element boxes.
o	Defines headings, sections, and containers for displaying detailed element information.

CSS (Cascading Style Sheets) is a style sheet language used to control the visual presentation of HTML elements. It allows designers and developers to create visually appealing and responsive web pages.
o	Enables styling of text, colours, layouts, and animations.
o	Supports responsive design for multiple devices.

JavaScript is a versatile programming language used to create interactive and dynamic web content. It works alongside HTML and CSS to bring functionality to web pages.
o	Enables interactive elements like clickable buttons or tables.
o	Supports real-time updates and data manipulation.
